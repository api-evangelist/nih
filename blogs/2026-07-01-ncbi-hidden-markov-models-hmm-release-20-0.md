NCBI released version 20.0 of its protein profile Hidden Markov Models, now totaling 18,950 HMMs with 497 new models added since release 19.0. The models support the Prokaryotic Genome Annotation Pipeline (PGAP) and can be searched against prokaryotic proteins using the HMMER analysis package to determine protein function.
